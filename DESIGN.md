# Design System: Modern Minimal

## Visual Posture
- **Tone**: Quiet, precise, software-native.
- **Typography**: Tight letter-spacing on display sizes (-0.02em), system fonts.
- **Borders**: Hairline borders, no shadows except for depth.
- **Palette**: Neutral foundations with a single electric-blue accent.

## Tokens (OKLch)
- `--bg`: `oklch(99% 0.002 240)`
- `--surface`: `oklch(100% 0 0)`
- `--fg`: `oklch(18% 0.012 250)`
- `--muted`: `oklch(54% 0.012 250)`
- `--border`: `oklch(92% 0.005 250)`
- `--accent`: `oklch(58% 0.18 255)`

## Components
- `MediaBetweenText`: The core interactive element, following the provided React spec.
